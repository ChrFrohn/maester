## SYNOPSIS

Returns hashtable of all policy GUIDs and if they are applied

## SYNTAX

```powershell
Get-PolicyStates [[-AntiphishPolicies] <Object>] [[-AntiphishRules] <Object>] [[-AntimalwarePolicies] <Object>]
 [[-AntimalwareRules] <Object>] [[-AntispamPolicies] <Object>] [[-AntispamRules] <Object>]
 [[-OutboundSpamPolicies] <Object>] [[-OutboundSpamRules] <Object>] [[-SafeLinksPolicies] <Object>]
 [[-SafeLinksRules] <Object>] [[-SafeAttachmentsPolicies] <Object>] [[-SafeAttachmentRules] <Object>]
 [[-ProtectionPolicyRulesATP] <Object>] [[-ProtectionPolicyRulesEOP] <Object>]
 [[-BuiltInProtectionRule] <Object>]
```
